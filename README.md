
## Setup
1. Folder `meditrust` into  XAMPP `htdocs/`
2. Create DB:
   - Open phpMyAdmin -> import `database/schema.sql`
3. DB creds in `app/config/database.php`
4. Open:
   - http://localhost/meditrust/public/

## Routes
- /emr
- /notifications
- /reports
- /telemedicine
- /feedback

## API 
- GET/POST /api/emr
- GET/POST /api/notifications
- GET/POST /api/visits
- GET/POST /api/telemedicine
- GET/POST /api/feedback
